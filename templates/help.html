{{ "Frequently Asked Questions"|t }}

🚀 {{ "Getting Started"|t }}

{{ "How do I create an account?"|t }}

{{ "Click the \"Get Started\" or \"Register\" button in the top navigation. Fill out the registration form with your name, email, password, and meal preferences (dietary restrictions, household size, cuisine variety). Accept the terms and click \"Create Account.\""|t }}

{{ "What dietary restrictions can I set?"|t }}

{{ "You can configure multiple dietary restrictions including Vegetarian, Vegan, Gluten-Free, Dairy-Free, and Nut-Free. The meal planning algorithm will respect these restrictions when generating meal plans."|t }}

{{ "Do I need to add recipes before generating a meal plan?"|t }}

{{ "Yes, you need to have favorited recipes before generating a meal plan. You can create your own recipes, import recipes via JSON, or favorite recipes from the community. We recommend having at least 15-20 favorites for best results."|t }}

📅 {{ "Meal Planning"|t }}

{{ "How many weeks does the system generate?"|t }}

{{ "The system automatically generates meal plans for all remaining weeks of the current month (from next week onwards). If you're at the end of the month, it will extend into the next month. Typically this generates 3-5 weeks of meals."|t }}

{{ "Can I see all generated weeks on the free tier?"|t }}

{{ "Yes. Free tier users have read-only access to all previously generated meal plans. Generating new meal plans requires a Premium subscription."|t }}

{{ "Why can't I regenerate the current week?"|t }}

{{ "The current week (Monday-Sunday range containing today) is locked to preserve your in-progress meals. This prevents accidentally replacing meals you've already started preparing. You can regenerate all future weeks individually or all at once."|t }}

{{ "What if I don't have enough recipes for all weeks?"|t }}

{{ "The system gracefully handles insufficient recipes by leaving empty meal slots. There's no minimum recipe count required. For example, with 10 favorites you might get Week 1 full and Week 2 partial with empty slots."|t }}

{{ "Will I get the same meal plan if I regenerate?"|t }}

{{ "No. Generation is non-deterministic - each regeneration produces different meal slot assignments even with the same favorite recipes. This allows you to regenerate until you find an arrangement you prefer."|t }}

{{ "How does accompaniment pairing work?"|t }}

{{ "When you create a main course recipe, you can mark it as accepting accompaniments and specify preferred types (rice, pasta, bread, fries, salad, vegetables). The algorithm will automatically pair compatible accompaniment recipes with your main courses when generating meal plans."|t }}

📖 {{ "Recipe Management"|t }}

{{ "What are the 6 recipe types?"|t }}

{{ "Recipes are categorized into 6 types: Appetizer (starters, salads), Main Course (entrees), Dessert (sweets), Accompaniment (sides like rice, pasta, bread), Beverage (cocktails, juices, smoothies, infusions), and Condiment (sauces, chutneys, marinades, vinaigrettes). Each day's meal plan can include an appetizer, main course with optional accompaniment, a beverage, a condiment, and a dessert."|t }}

{{ "How do I import recipes from JSON?"|t }}

{{ "Go to the Recipe Import page, drag and drop JSON files (max 10MB each, up to 20 files per batch), and click Import. The system validates each recipe against the schema and provides a summary showing successful imports, failed recipes, and duplicates. Check our JSON Schema documentation for the required format."|t }}

{{ "Can free tier users save recipes?"|t }}

{{ "Saving recipes to your library requires a Premium subscription. Free tier users can browse community recipes but cannot save, edit, or create their own."|t }}

{{ "Are dietary restrictions automatically detected?"|t }}

{{ "No. All recipe fields including dietary restrictions and preferences must be explicitly set by users when creating or importing recipes. The system does not automatically detect or infer these attributes from recipe content."|t }}

{{ "Can I edit recipes I imported from JSON?"|t }}

{{ "Yes. Imported recipes are stored in your recipe library just like manually created recipes. You can edit, delete, share, or favorite them like any other recipe."|t }}

👥 {{ "Community"|t }}

{{ "How do I share a recipe with the community?"|t }}

{{ "When creating or editing a recipe, check the \"Share with community\" toggle. Shared recipes become publicly visible in community search, allowing other users to discover, favorite, and include them in their meal plans."|t }}

{{ "What happens when someone favorites my shared recipe?"|t }}

{{ "When a user favorites your recipe, they can include it in their meal plans. You are not notified when someone favorites your recipe. If you delete a shared recipe, it's automatically removed from all users' favorites."|t }}

{{ "What are recipe snapshots?"|t }}

{{ "When meal plans are generated, the system creates complete snapshots/copies of all referenced recipes (including community recipes). This ensures meal plan integrity if the original recipe owner deletes or modifies their recipe. Historical meal plans always retain the original recipe data."|t }}

{{ "Can I rate and review community recipes?"|t }}

{{ "Yes. All users (free and premium) can rate and review community recipes. Ratings help maintain quality - low-rated recipes are buried in search results. There's no pre-approval required; the community self-moderates through feedback."|t }}

{{ "Can free tier users share recipes?"|t }}

{{ "No. Creating, editing, and sharing recipes requires a Premium subscription. Free tier users can browse and view community recipes in read-only mode."|t }}

💳 {{ "Account & Billing"|t }}

{{ "What's the difference between Free and Premium?"|t }}

{{ "Free: Read-only access. Browse community recipes, view your existing recipes and meal plans, preview kitchen tools.
Premium ($4.99/mo or $47.90/yr): Create and edit recipes, generate personalized meal plans, use step-by-step kitchen tools, bulk import recipes, priority support, early access to features."|t }}

{{ "How do I upgrade to Premium?"|t }}

{{ "Click any \"Upgrade to Premium\" button throughout the app, or go to your Profile/Settings page and select the subscription option. Choose monthly ($4.99/mo) or annual ($47.90/yr, 20% savings) billing. Enter your payment information and confirm."|t }}

{{ "Can I cancel my subscription anytime?"|t }}

{{ "Yes. Cancel anytime through your account settings. You'll retain premium access until the end of your paid period, then automatically revert to the free tier. No refunds are provided for partial months or unused portions of annual subscriptions."|t }}

{{ "What happens to my data when I downgrade to free?"|t }}

{{ "When downgrading, your account switches to read-only mode. All your recipes and meal plans are preserved and remain viewable. You can no longer create, edit, generate, or import — upgrade anytime to restore write access."|t }}

{{ "Do subscriptions auto-renew?"|t }}

{{ "Yes. Subscriptions automatically renew at the end of each billing period (monthly or annual). You'll receive an email notification before renewal. Cancel anytime to prevent renewal."|t }}

{{ "What payment methods do you accept?"|t }}

{{ "We accept all major credit cards (Visa, MasterCard, American Express, Discover) processed through secure PCI-DSS compliant payment processors. We do not store full card numbers."|t }}

🔧 {{ "Technical Support"|t }}

{{ "Which browsers are supported?"|t }}

{{ "We support modern browsers: Chrome 90+, Firefox 88+, Safari 14+, and Edge 90+. For best experience, keep your browser updated to the latest version."|t }}

{{ "Is the app mobile-responsive?"|t }}

{{ "Yes. imkitchen is a Progressive Web App (PWA) designed mobile-first with touch-optimized interfaces, week carousel navigation, and responsive layouts. It works on iOS Safari, Android Chrome, and can be installed as a standalone app."|t }}

{{ "Why can't I log in?"|t }}

{{ "Common login issues: (1) Check your email/password are correct, (2) Clear browser cache and cookies, (3) Try password reset if forgotten, (4) If account is suspended by admin, you won't be able to log in - contact support."|t }}

{{ "How long does meal plan generation take?"|t }}

{{ "Multi-week generation completes in under 5 seconds for up to 5 weeks (P95 performance). If generation takes longer, check your internet connection or contact support."|t }}

{{ "What if my JSON import fails?"|t }}

{{ "The import summary shows which recipes failed and why. Common issues: (1) Invalid JSON format, (2) Missing required fields, (3) File size exceeding 10MB, (4) Duplicate recipes. Review the JSON Schema documentation and ensure your files match the required structure. Only 100% valid recipes are imported."|t }}

{{ "Can I use the app offline?"|t }}

{{ "Offline recipe access is supported. You can view previously loaded recipes and meal plans offline. Generating new meal plans, syncing data, and community features require an internet connection."|t }}
