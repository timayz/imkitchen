Browser Support

imkitchen is built as a Progressive Web App (PWA) optimized for modern browsers. For the best experience, please ensure your browser meets the minimum requirements below.

Supported Browsers

Mobile

  • iOS Safari 14+ iPhone, iPad (iOS 14 or later)
  • Android Chrome 90+ Android 7.0+ recommended

Desktop

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

Feature Support Matrix

Feature Chrome/Android Firefox Safari/iOS
Service Worker
Offline Caching
Background Sync
Wake Lock API 16.4+
Web Push Notifications 16.4+

Note: Background Sync is not supported on iOS Safari and Firefox. Your changes will sync automatically when you open the app on these browsers.

Known Limitations

iOS Safari

  • Background Sync not supported - changes sync when app is open
  • Wake Lock API supported on iOS 16.4+
  • Web Push notifications require iOS 16.4+ and user gesture

Firefox

  • Background Sync not supported - changes sync when app is open
  • Wake Lock API not supported - screen may sleep during cooking

How to Check Your Browser Version

Chrome (Desktop/Android)

Settings → About Chrome → Version displayed at the top

Safari (iOS)

Settings → General → About → Software Version (iOS version indicates Safari version)

Firefox

Menu → Help → About Firefox → Version displayed

Need to Upgrade?

If your browser doesn't meet the minimum requirements, please update to the latest version for the best experience and security.
