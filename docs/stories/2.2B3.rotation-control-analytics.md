### Technical Architecture Context

**Rotation Control & Analytics Logic**

**Reset Functionality**:
```go
// Rotation Reset with State Preservation
type RotationResetRequest struct {
    ConfirmReset     bool `json:"confirmReset" validate:"required"`
    PreservePatterns bool `json:"preservePatterns" default:"true"`
    PreserveFavorites bool `json:"preserveFavorites" default:"true"`
}

func (r *RotationService) ResetRotationCycle(userID uuid.UUID, req RotationResetRequest) error {
    if !req.ConfirmReset {
        return errors.New("reset confirmation required")
    }
    
    // Clear rotation history but preserve user preferences
    return r.clearRotationState(userID, req)
}
```

**Analytics Calculations**:
```go
// Comprehensive Rotation Analytics
type RotationAnalytics struct {
    VarietyScore        float64            `json:"varietyScore"`        // 0-100 scale
    ComplexityTrends    []ComplexityTrend  `json:"complexityTrends"`    // Weekly trends
    FavoritesFrequency  map[string]int     `json:"favoritesFrequency"`  // Recipe frequency
    WeeklyPatterns      []WeeklyAnalysis   `json:"weeklyPatterns"`      // Pattern adherence
    RotationEfficiency  float64            `json:"rotationEfficiency"`  // Algorithm performance
}

func (a *AnalyticsService) CalculateVarietyScore(userID uuid.UUID, weeks int) float64 {
    // Complex variety calculation using Shannon entropy and rotation coverage
    return a.shannonEntropyScore() * a.rotationCoverageScore()
}
```

**API Specifications**

**New API Endpoints**:
```yaml
# Rotation Control
POST /api/v1/users/rotation/reset:
  summary: Reset user's rotation cycle with confirmation
  requestBody:
    confirmReset: boolean (required)
    preservePatterns: boolean (default true)  
    preserveFavorites: boolean (default true)
  responses:
    200: Reset completed successfully
    400: Invalid confirmation or request
    
# Analytics & Reporting
GET /api/v1/users/rotation/stats:
  summary: Get comprehensive rotation analytics
  parameters:
    - weeks: integer (analysis period, default 12)
    - includeDetailed: boolean (detailed breakdown)
  responses:
    200: Complete rotation analytics object

GET /api/v1/users/rotation/export:
  summary: Export rotation data for analysis
  parameters:
    - format: string (json|csv)
    - dateRange: string (ISO date range)
  responses:
    200: Exported rotation data file
```
