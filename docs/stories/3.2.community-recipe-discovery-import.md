# Story 3.2: Community Recipe Discovery & Import

## Status

Complete - Ready for Production

## Story

**As a** recipe explorer,  
**I want** to discover and import highly-rated community recipes,  
**so that** I can expand my recipe collection with validated, quality options.

## Acceptance Criteria

1. Community recipe browse interface with search and filtering capabilities
2. Recipe import from community to personal collection with one-tap functionality
3. Trending recipes section highlighting recently popular community additions
4. Category-based recipe discovery (vegetarian, quick meals, comfort food, etc.)
5. User-generated recipe tags and community-driven categorization
6. Recipe attribution showing original contributor and community metrics

### Technical Constraints
**Performance Requirements**:
- Recipe import operation: <2 seconds completion time
- Community recipe discovery: <1 second initial load, lazy loading for subsequent pages
- Tag autocomplete: <200ms response time for suggestions
- Trending algorithm: Real-time updates without blocking user interactions

**Business Rules**:
- Recipe attribution must be preserved during import (contributor credit)
- Maximum 10 user-generated tags per recipe to prevent spam
- Import rate limiting: Maximum 20 imports per hour per user
- Tag moderation: Community voting system for tag validation
- Recipe import creates new personal recipe, preserving original community recipe

### Error Handling & Recovery Procedures
**Import Operation Failures**:
- **Duplicate Recipe**: Show merge options or rename suggestion
- **Import Service Down**: Queue import for retry with offline notification
- **Attribution Missing**: Require manual contributor acknowledgment
- **Storage Limit**: Clear guidance on recipe management and deletion options

**Discovery & Search Errors**:
- **Search Service Unavailable**: Fallback to cached trending recipes
- **Category Loading Failed**: Show simplified category list from local storage
- **Tag Service Down**: Allow manual tag entry without autocomplete
- **Empty Search Results**: Helpful suggestions for broadening search criteria

### Security Considerations
**Import System Security** [Source: architecture/15-security-and-performance.md]:
- Recipe import permission validation (authenticated users only)
- Rate limiting on import operations to prevent abuse
- Tag content validation to prevent XSS in user-generated tags
- Attribution integrity to prevent false contributor claims
- Community recipe access control for private/public recipes

### Caching Strategy
**Discovery & Import Performance** [Source: architecture/11-backend-architecture.md]:
- Trending recipes cached for 30 minutes in Redis
- Category-based recipe lists cached for 1 hour
- Tag autocomplete suggestions cached for 4 hours
- Recipe import operations bypass cache to ensure real-time updates
- User import history cached for 15 minutes
